This project has two main functions. First is the CV maker and second is the Cv Reader.
Let us say that someone want to make a CV but does not know how to do it. They can give the required information to it and get beautifully made CVs.
For the second one, If someone has many CVs and want to shortlist them on the base of like Skills, Experience etc they can do it by using this and can save alot of time.
